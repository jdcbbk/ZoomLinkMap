#Zoom Link Map

a tool to analyze the link map file that is generated by clang. Using this tool you can know the space of each part for app.

##Install

```
sudo pip install zoomlinkmap
```

##Usage

```
zoomLinkMap [-h] path
```

example:

```
zoomLinkMap TimeUI-LinkMap-normal-armv7.txt
```
